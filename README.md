Trabajo Práctico N° 1

Este repositorio contiene la resolución de los ejercicios prácticos integradores correspondientes al primer trabajo práctico de la materia.
Descripción de los ejercicios:

tp1_ejercicio1.py (Sistema de Transporte):
- Se modeló un sistema utilizando Programación Orientada a Objetos.
- Se define una clase base Vehiculo y dos subclases: Auto y Moto.
- Se demuestra el uso de herencia (reutilizando atributos del padre) y polimorfismo, recorriendo una lista de objetos y ejecutando el método describir() para obtener información específica de cada vehículo.

tp1_ejercicio2.py (Cálculo de Áreas):
- Se implementó una clase base Figura con un método area() que retorna 0 por defecto.
- Se crearon las subclases Rectangulo y Circulo, donde se sobrescribe el método area() con sus fórmulas matemáticas correspondientes.
- Se utilizó la librería estándar math para obtener el valor preciso de PI y el operador ** para potencias.
- El polimorfismo se verifica al imprimir el informe de áreas desde una lista de figuras mixtas.

Instrucciones de ejecución:  Para correr los archivos correctamente, seguir estos pasos:
- Asegurate de tener instalado Python 3.13 o superior.
- Abrí una terminal o consola en la carpeta trabajo-practico-I/.
- Ejecutá los archivos con los siguientes comandos: Para el Ejercicio 1: python tp1_ejercicio1.py -- Para el Ejercicio 2: python tp1_ejercicio2.py
